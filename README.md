Colorful format iterable

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
spare = ">=0.0.1"
```

and this to your crate root:

```rust
extern crate spare;
```

## Compatibility

The `spare` crate is tested for rustc 1.48 and greater.
